# WAD-HW4
# Backend setup

## Clone the repository
```
git clone https://github.com/henrilipping/WAD-HW4.git
```

## Navigate to the backend
```
cd backend
```

## Install dependencies
```
npm install
```

## Run the project
```
npm start
```
or
```
npm start
```


# Frontend setup

## Open new terminal
## Navigate to the frontend
```
cd frontend
```

## Install dependencies
```
npm install
```

## Compiles and hot-reloads for development
```
npm run serve
```

## Compiles and minifies for production
```
npm run build
```
